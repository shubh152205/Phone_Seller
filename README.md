# Baba The Hi-Tech (Phone Seller)

A premium e-commerce platform for selling mobile phones and managing repair services. Built with a focus on high-performance, modern aesthetics, and seamless user experience.
demo: https://phone-seller.vercel.app/

## 🚀 Features

- **Premium UI/UX**: Modern, responsive design with glassmorphism and smooth animations.
- **Product Catalog**: Browse and view detailed mobile phone specifications.
- **Repair Services**: Dedicated section for booking and tracking phone repairs.
- **Admin Dashboard**: Comprehensive management tool for:
    - Inventory Management
    - Repair Order Tracking
    - Customer Directory
    - Analytics & Insights
- **Dark Mode**: Fully supported dark and light modes.
- **Interactive Components**: Using Recharts for data visualization in the admin panel.

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Custom design system)
- **Charts**: Recharts
- **State Management**: React Context API

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shubh152205/Phone_Seller.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Phone_Seller
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 🌐 Deployment

This project is optimized for deployment on platforms like **Vercel** or **Netlify**.

### Deploying to Vercel (Recommended)

1. Push your code to GitHub (Done).
2. Go to [Vercel](https://vercel.com/) and sign in with GitHub.
3. Click "New Project" and import the `Phone_Seller` repository.
4. Vercel will automatically detect Vite settings. Click **Deploy**.
